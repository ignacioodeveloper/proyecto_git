EJERCICIOS COMANDO GIT

  1. git init -> crear repositorio
  2. git status -> ver el estado del git si todo se ejecuto con exito git status -sb -> short form
  3. git add . -> agregar todos ( . ) los archivos al repositorio al guardar en VS hay que agregar git add (nombre del archivo o ( . )) para que se guarden los cambios
  4. git commit -m "mensaje_claro" -> guardar un respaldo del proyecto
  5. git log -> ver git ejecutados o creados git log --oneline
  6. git remote add origin "link_proyect" -> agrega una conexion remota
  7. git branch -M main -> modifica el nombre de la rama
  8. git push -u origin main -> lleva el directorio local a la nube "empuja | push"
